О ИГРЕ
Суть игры заключается в преодоление препятствий игровым персонажем - котиком по имени Барсик. Препятствие представлены в виде лимонных деревьев (коты не любят цитрусовых). Чтобы перепрыгнуть через препятствие, игрок должен нажать любую кнопку на клавиатуре. Игрок, набравший более N количества баллов за N количество времени выигрывает. Игрок, набравший менее N баллов за N времени, проигрывает. Приятного времяпрепровождения :)
